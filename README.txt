CONTENUTI:

    --> run-theknife.bat: file eseguibile batch per Windows
    --> run-theknife.sh: file eseguibile shell per Linux e macOS
    --> src/: codice sorgente del progetto
    --> data/: file dati CSV dell'applicazione
    --> doc/: documentazione e manuali

Doppio click su "run-theknife.bat" (Windows) o "run-theknife.sh" (Linux/macOS) dalla directory principale del progetto.
